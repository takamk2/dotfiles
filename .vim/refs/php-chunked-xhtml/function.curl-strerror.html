curl_strerror

(PHP 5 >= 5.5.0)

curl_strerror — エラーコードの説明を返す

説明

string curl_strerror ( int $errornum )

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

errornum

定数 CURLE_* のいずれか。

返り値

エラーの説明を返します。無効なエラーコードを指定した場合は NULL を返します。
