curl_share_init

(PHP 5 >= 5.5.0)

curl_share_init — 共有 curl ハンドルを初期化する

説明

resource curl_share_init ( void )

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

"cURL Share Handle" 型のリソースを返します。
