curl_escape

(PHP 5 >= 5.5.0)

curl_escape — 指定した文字列を URL エンコードする

説明

string curl_escape ( resource $ch , string $str )

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

ch

curl_init() が返す cURL ハンドル。

str

返り値

エスケープした文字列を返します。失敗した場合に FALSE を返します。
