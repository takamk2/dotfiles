cli_get_process_title

(PHP 5 >= 5.5.0)

cli_get_process_title — 現在のプロセスのタイトルを返す

説明

string cli_get_process_title ( void )

CLI でしか使えません。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

現在のプロセスのタイトルを文字列で返します。エラー時には NULL を返します。
